Testing

Tests are implemented using Python’s pytest module and live MySQL and API servers using docker.

The docker containers are started upon initiating tests using pytest and all the tests are run against the live MySQL and API servers.

Example run:

currency-converter-api> cd backend
currency-converter-api\backend> pytest -s -v
