async convert_rate(from_curr_code: str, to_curr_code: str, amount: float, on_date: Optional[str] = None) → Dict[str, Any]

Convert currency from one to another based on the rate on the given date.

If a date is not provided, the date from the last update is used.

Parameters

  • from_curr_code (required) – Currency code to convert from.

  • to_curr_code (required) – Currency code to convert to.

  • amount (required) – Amount to be converted

  • on_date (optional) – Currency rates from this date will be used if provided.

Raises

ValueError – Propogated from exception stack

Returns

dict of Currency rate converted from -> to, base currency and last updated date of the currency from the external API

db.utils.wait_for_db(conn_settings: Dict[str, Any], sleep: int = 5, extra_sleep: int = 0) → bool

Attempts to connect to the database indefinitely, in case the database is not yet up.

While attempting to connect, displays reason for delay in connecting.

Parameters

  • conn_settings (required) – Dictionary containing the setting:value pairs

  • sleep (optional, default=5) – Number of seconds between reconnection attempts.

  • extra_sleep (optional) – Extra seconds to sleep after a successful connection.
