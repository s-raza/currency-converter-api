DB Updates

Updates the database periodically with the latest currency rates from the external source API

async start_updates.execute(freq_mins: int) → None

Execute the DB update.

Parameters

freq_mins (required) – Interval in minutes for the update schedule.

Returns

None

start_updates.main() → None

Main entry point for the currency rates DB updater.

The interval in seconds for the update schedule can be provided on the command line using the -m switch. If it is not provided, the value of UPDATER__FREQUENCY from the .env file is used via the pydantic settings module.
