Usage
=====

The currency converter is implemented using individual services running inside
docker containers.

The React GUI for the currency converter can be accessed after orchestrating these
containers either through :doc:`Docker <usage/docker/index>` or
:doc:`Kubernetes <usage/k8s/index>`
