External Source API
===================

The latest rates are retrieved from
`Exchange Rate <http://api.exchangerate.host/latest/>`_

The :ref:`Update Server <containers-updater>` prepetually retrieves latest
currency rates from `Exchange Rate <http://api.exchangerate.host/latest/>`_ and
updates the database in the :ref:`MySQL Database Server <containers-database>`
