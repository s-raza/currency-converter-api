Run Services using KubernetesÔÉÅ

There is an option to run the currency converter using Kubernetes.

Minikube can be used to run the kubernetes cluster locally, which requires installing and intializing Minikube locally on Docker Desktop for Windows or on Linux

General kubectl commands can be used to run the kubernetes cluster on Minikube.

Alternatively a Powershell Script for Windows systems is provided to manage the Kubernetes cluster running in Minikube.

In both cases Minkube should be installed and initiated before using any of the above two methods.

To install Minikube for your OS click here for the official instructions
